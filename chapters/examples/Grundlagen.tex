\section{Abstrakte Argumentations Formalismen}
\label{sec:af}

In diesem Kapitel werden \emph{Abstrakte Argumentations Formalismen}~\cite{dung:1995:af},
Semantiken und labelling basierte Sematiken vorgestellt. 
Außerdem wird das Konzept der \emph{bedingten Unabhängigkeit} eingeführt. 
Schlussendlich werden \emph{Quantifizierte Boolesche Formeln} vorgestellt. 

\emph{Abstrakte Argumentations Formalismen} (AF) sind ein zentrales Konzept
der formalen Argumentationstheorie. Sie wurden von Dung~\cite{dung:1995:af}
eingeführt, um das grundlegende Prinzip menschlicher Argumentation formal zu erfassen und maschinell nutzbar zu machen.
Der Grundgedanke von AF ist es, Argumente als abstrakte Einheiten zu betrachten, die durch eine Angriffsrelation miteinander verbunden sind.
Damit ist die Akzeptanz eines Arguments immer von der Auswertung anderer Argumente abhängig und nicht von der inneren Struktur des Arguments selbst. 

\begin{Definition}
    \label{def:af}
Ein \emph{Abstrakter Argumentations Formalismus} (AF) ist ein Paar $F = (A, R)$, 
wobei $A$ eine Menge von Argumenten und
 $R \subseteq A \times A$ eine binäre Angriffsrelation auf $A$ ist.
 Für ein AF $F = (A, R)$ repräsentiert $A_F$ die Menge der Argumente von $F$ und $R_F$ die Angriffsrelation von $F$.
\end{Definition}

\section{Semantiken}
\label{sec:af-semantiken}

Die Frage, welche Argumente oder Argumentmengen in einem Argumentationsszenario als überzeugend gelten sollen, 
wird mit dem Konzept der Extension beantwortet.
Eine Extension ist eine Menge von Argumenten, die zusammen gewünschte Eigenschaften erfüllen. 
Sie also aufgrund ihrer strukturellen Eigenschaften überzeugend macht~\cite{dung:1995:af}. 

\begin{Definition}
      \label{def:extension}
      Eine Extension $E \subseteq A$ ist eine Auswahl der Argumentmenge $A$, die bestimmte Eigenschaften erfüllt.
\end{Definition}

\begin{Definition}
      \label{def:semantik}
      Eine Semantik $\sigma$ ist eine Funktion, die jedem AF F eine Menge an Extensionen zuordnet. Es gilt: $\sigma(F) \subseteq 2^{A}$
\end{Definition}
Test \textcite{darwiche:1997:logical-ci}.
Rapberger Paper \cite{bluemel:2025:independence-aba}
Rienstra Paper \cite{rienstra:2020:independence-af}\\
Dabei lässt zwischen extension-basierten und labelling-basierten Semantiken unterscheiden.
Extension-basierte Semantiken definieren die Akzeptanz von Argumenten durch die Auswahl von Mengen von Argumenten, 
die bestimmte Eigenschaften erfüllen.
Labelling-basierte Semantiken hingegen ordnen jedem Argument ein Label zu, das seinen Akzeptanzstatus festhält.

Im Folgenden wird der Unterschied der beiden Ansätze anhand einiger grundlegender Sematiken erläutert.

\subsection{Extension-basierte Semantiken}

Die Definition einer extension-basierten Sematik basiert lediglich auf der Festlegung von bestimmten Eigenschaften, die eine Menge von 
Argumenten erfüllen muss, um als akzeptabel zu gelten.

Angriff
\begin{Definition}
      \label{def:attack-single-argument}
      Sei $F = (A, R)$ ein AF. Ein Argument $b \in A$ heißt \emph{Angreifer} auf ein Argument $a \in A$, genau dann wenn $(b, a) \in R$ gilt.
      Wir schreiben: $b \rightarrow a$ für $(b, a) \in R$. Und definieren die Menge aller Angreifer von $a$ als $a^{-} = \{b \in A \mid b \rightarrow a\}$.
\end{Definition}
Für eine Menge von Argumenten formalisieren wir das Konzept des Angriffs wie folgt:
\begin{Definition}
      \label{def:attack-set-arguments}
      Sei $F = (A, R)$ ein AF. Ein Argument $b \in A$ heißt \emph{Angreifer} auf eine Menge von Argumenten $S \subseteq A$, 
      genau dann wenn es ein Argument $a \in S$ gibt, so dass $(b, a) \in R$ gilt.
      Wir schreiben: $b \rightarrow S$ für $b \in A$, $a \in S$, $S \subseteq A$ und $(b, a) \in R$. 
      Die Menge aller Angreifer von $S$ definieren wir als $S^{-} = \{b \in A \mid b \rightarrow S\}$.
\end{Definition}

Wir fassen formal zusammen.

\begin{Definition}
      \label{attack-object-subject}
      Sei $F = (A, R)$ ein AF, eine Menge $S$ mit $S \subseteq A$ und $a$ ein Argument mit $a \in A$. Wir schreiben:
\begin{itemize}
      \item $a^{+} = \{b \in A \mid a \rightarrow b\}$
      \item $a^{-} = \{b \in A \mid b \rightarrow a\}$
      \item $S^{+} = \{b \in A \mid a \rightarrow b \text{ für ein } a \in S\}$
      \item $S^{-} = \{b \in A \mid b \rightarrow a \text{ für ein } a \in S\}$
\end{itemize}
\end{Definition}

Neben des expliziten Konzepts des Angriffs, transportiert die Angriffsrelation implizit auch die Idee der \emph{Verteidigung}.
Soll ein einzelnes Argument Teil unserer ausgewählten Menge sein, so sollte es
gegen Angriffe von außen geschützt sein. Diese Eigenschaft definieren wir wie folgt:
\begin{Definition}
    \label{def:defense}
    Sei $F = (A, R)$ ein AF. Ein Argument $a \in A$ heißt \emph{verteidigt} durch eine Menge $S \subseteq A$, 
    genau dann wenn für jedes Argument $b \in A$ mit $b \rightarrow a$ gilt, dass es ein Argument $c \in S$ gibt, so dass $c \rightarrow b$ gilt.

\end{Definition}


Basis aller Semantiken ist die \emph{Konfliktfreiheit}. Eine Menge an Argumenten soll ein Argumentationsszenario nur "überleben", 
wenn es in sich selbst stimmig ist \cite{handbook__of_formal_argumentation__vol__3:2024}.
\begin{Definition}
    \label{def:conflict-free}
    Sei $F = (A, R)$ ein AF. Eine Menge $S \subseteq A$ heißt \emph{konfliktfrei}, genau dann wenn es keine Argumente $a, b \in S$ gibt, 
    so dass $a \rightarrow b$ gilt.
\end{Definition}
Insbesondere ist damit ein Argument, das sich selbst angreift, nicht \emph{konfliktfrei}.

\begin{Definition}
      \label{range}
      Sei $F = (A, R)$ ein AF und $S \subseteq A$. Die \emph{Reichweite} von $S$ ist definiert als $S \cup S^{+}$.
\end{Definition}

\begin{Definition}
      \label{semantic-extensions}
      Sei $F = (A, R)$ ein AF, $S \subseteq A$ und $S$ \emph{konfliktfrei}. $S$ heißt:
\begin{itemize}
      \item \emph{admissible}, genau dann wenn jedes Argument in $S$ durch $S$ verteidigt wird.
      \item \emph{complete}, genau dann wenn jedes Argument, das durch $S$ verteidigt wird, in $S$ enthalten ist.
      \item \emph{preferred}, genau dann wenn $S$ inhaltsmaximal unter den admissible Mengen ist.
      \item \emph{stable}, genau dann wenn $S^{+} = A \setminus S$ gilt.
\end{itemize}
\end{Definition}

\subsection{Labelling-basierte Semantiken}
\label{sec:af-labelling}
Eine weitere Möglichkeit, die Akzeptanz von Argumenten zu bestimmen, 
ist der labelling-basierte Ansatz, 
der von Caminada~\cite{caminada:2009:logical-account} eingeführt wurde.
\begin{Definition}
  \label{def:labelling}
  Sei $F = (A, R)$ ein AF. Ein \emph{labelling} von $F$ ist eine Funktion
  $\mathcal{L} : A \rightarrow \{\text{in}, \text{out}, \text{und}\}$, die jedem Argument
  eines von drei Labeln zuordnet: \texttt{in}, \texttt{out} oder \texttt{und}.
\end{Definition}

Wir verwenden $\text{in}(\mathcal{L})$ für $\{a \mid \mathcal{L}(a) = \text{in}\}$, 
$\text{out}(\mathcal{L})$ für $\{a \mid \mathcal{L}(a) = \text{out}\}$ 
und $\text{und}(\mathcal{L})$ für $\{a \mid \mathcal{L}(a) = \text{und}\}$.

\begin{Definition}
      \label{def:complete-labelling}
      Sei $F = (A, R)$ ein AF und $\mathcal{L} : A \rightarrow \{\text{in}, \text{out}, \text{und}\}$ ein $labelling$
      von $F$. $\mathcal{L}$ heißt \emph{complete}, wenn für jedes Argument $a \in A$ gilt:
      \begin{itemize}
            \item $\mathcal{L}(a) = \text{in}$ gdw. für jedes $b \in A$ mit $(b, a) \in R$ $\mathcal{L}(b) = \text{out}$ gilt.
            \item $\mathcal{L}(a) = \text{out}$ gdw. es ein $b \in A$ mit $(b, a) \in R$ gibt, für das $\mathcal{L}(b) = \text{in}$ gilt.
      \end{itemize}
\end{Definition}

\section{Conditional Independence}
\label{sec:conditional-independence}
      Der Begriff der \emph{conditional independence} ist ein zentrales Konzept in der Wahrscheinlichkeitstheorie.
      Er beschreibt die Unabhängigkeit von zwei Zufallsvariablen unter der Bedingung, 
      dass eine andere dritte Variable einen bestimmten Wert annimmt.

      Die semi-graphoiden Axiome, nach Pearl~\cite{pearl:2010}, sind eine Reihe von Eigenschaften, 
      die die konditionale Unabhängigkeitsrelation erfüllt.\par Sie lauten wie folgt:
      \begin{itemize}
            \item Symmetrie: $A \perp B \mid C \implies B \perp A \mid C$
            \item Dekomposition: $A \perp B \cup D \mid C \implies A \perp B \mid C$ 
            \item Schwache Vereinigung: $A \perp B \cup D \mid C \implies A \perp B \mid C \cup D$ 
            \item Kontraktion: $A \perp B \mid C \land A \perp D \mid C \cup B \implies A \perp B \cup D \mid C$
      \end{itemize}


\section{Quantified Boolean Formulas}
\label{sec:qbf}
      \emph{Quantified boolean formulas} (QBF) sind eine Erweiterung der klassischen Aussagenlogik, 
      bei der Quantoren über boolesche Variablen verwendet werden.
      \begin{Definition}
      \label{def:qbf}
      Sei $A$ ein boolescher Ausdruck. Ein \emph{QBF} ist 
      \end{Definition}

      Für eine endliche Menge aussagenlogischer Variablen
      \(V=\{v_1,\dots,v_n\}\) schreiben wir
      \[
      \forall V\,\varphi
      \]
      als Abkürzung für
      \[
      \forall v_1\dots \forall v_n\,\varphi.
      \]
      Analog verwenden wir \(\exists V\).

      Ebenso nutzen wir für zwei endliche Mengen aussagenlogischer Variablen $X=\{x_1,\dots,x_n\}, Y=\{y_1,\ldots,y_n\}$
      \[
      \forall X,Y \varphi
      \quad \text{als Abkürzung für}\quad
      \forall X \forall Y \varphi
      \]
